### 상세 배분 계획

#### 커뮤니티 트레저리 (30%)
- 목적: DAO 거버넌스 통과 제안의 제작비 지원, 커뮤니티 이벤트
- 집행: DAO 투표로만 사용 가능
- 운영: 멀티시그 지갑 (Phase 2) → 온체인 자동 집행 (Phase 3)

#### 콘텐츠 리워드 풀 (25%)
- 목적: 기여자 보상 (제안, 거버넌스 참여, 팬 콘텐츠)
- 배분 방식: 기여도 기반 자동 분배
- 배분 일정: 5년에 걸쳐 분할 지급 (체감 방식)

```
Year 1: 8,000,000 LORE (32%)
Year 2: 6,000,000 LORE (24%)
Year 3: 5,000,000 LORE (20%)
Year 4: 4,000,000 LORE (16%)
Year 5: 2,000,000 LORE (8%)
```

#### 창립팀 (20%)
- 4년 베스팅, 1년 클리프
- 클리프 후 매월 1/36씩 선형 언락

#### 얼리 컨트리뷰터 (15%)
- Phase 0~1 초기 커뮤니티 기여자, 투자자
- 2년 베스팅, 6개월 클리프

#### 리저브 (10%)
- 목적: 거래소 상장, 파트너십, 법적 비용
- DAO 과반수 동의 없이 사용 불가

## 토큰 유틸리티

### 1. 거버넌스 (핵심)
```
보유량에 비례한 투표권
  - 기본: 1 LORE = 1표
  - Phase 3: 쿼드라틱 투표 적용 (√LORE = 표)

제안 스테이킹:
  - Tier 3 제안: 500 LORE 스테이킹
  - Tier 2 제안: 1,000 LORE 스테이킹
  - Tier 1 제안: 5,000 LORE 스테이킹
  - Canon 격상: 10,000 LORE 스테이킹
  - 거버넌스 규칙 변경: 50,000 LORE 스테이킹
```

### 2. 멤버십 티어 접근권
```
Tier 1 · Viewer     : 토큰 불필요 (무료 시청/커뮤니티)
Tier 2 · Contributor: 100 LORE 보유 (제안 투표권)
Tier 3 · Lorekeeper : 5,000 LORE 스테이킹 (가중 투표, 위원회 후보)
Tier 4 · Founding   : 창립 배분 보유자 (Core Veto 협의 참여)
```

### 3. 독점 콘텐츠 접근
- 메이킹 영상, 감독 코멘터리: 100 LORE 보유
- 미공개 설정집, 프리프로덕션 자료: 1,000 LORE 보유
- 제작진과의 Q&A, 캐릭터 네이밍 참여: 5,000 LORE 스테이킹

### 4. 커뮤니티 캐릭터 소유권 (NFT 연동, Phase 3)
- 커뮤니티가 제안하고 Canon 편입된 캐릭터의 원작자에게 NFT 발행
- 해당 캐릭터가 등장하는 콘텐츠 수익의 일부를 NFT 홀더에게 배분

## 토큰 획득 방법

### 기여도 기반 보상 (콘텐츠 리워드 풀에서 지급)

| 행동 | 보상 (예시) |
|------|-------------|
| Lore 제안 제출 (승인됨) | 500 ~ 5,000 LORE |
| Lore 제안 제출 (부결됨) | 50 LORE (참여 보상) |
| 거버넌스 투표 참여 | 10 LORE / 투표 |
| Lorekeeper 역할 수행 | 분기당 5,000 LORE |
| 팬 콘텐츠 제작 (선정됨) | 200 ~ 2,000 LORE |
| 번역, 요약 등 기여 | 50 ~ 500 LORE |
| 버그/오류 제보 (설정 충돌) | 100 LORE |

유통량 감소를 통한 장기적 토큰 가치 보호:

| 소각 트리거 | 소각량 |
|-------------|--------|
| 악의적 제안 패널티 (스팸/충돌 고의) | 스테이킹 전액 소각 |
| 제안 철회 (토론 중) | 스테이킹의 50% |
| 멤버십 구독료 (Phase 3 도입 시) | 구독료의 10% |
| 플랫폼 수익 환원 (분기별) | 수익의 5%를 매수 후 소각 |

## 거버넌스와 토큰의 연결 흐름

```
콘텐츠 공개 (영상)
      ↓
시청자 반응 → 커뮤니티 토론
      ↓
LORE 토큰 보유자가 Lore Proposal 제출
      ↓
스테이킹 (제안 보증금)
      ↓
Lorekeeper 검토 → 공개 투표
      ↓
    승인 시                부결 시
      ↓                     ↓
Canon 편입            Tier 4 아카이브
스테이킹 반환         스테이킹 반환
제안자 보상 지급       소액 참여 보상
      ↓
콘텐츠 팀이 Canon 설정으로 차기 영상 제작
      ↓
트레저리에서 제작비 지원 (DAO 승인 시)
```
