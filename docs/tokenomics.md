## 비즈니스 모델 한 줄 요약

**"시나리오를 쓰고 → DAO로 투표하고 → AI가 영상을 만든다. 더 많이 스테이킹할수록 더 좋은 영상이 나온다."**

## 핵심 가치 흐름

```
커뮤니티 시나리오 제출 (LIP)
         ↓
  LORE 스테이킹 (보증금 + 품질 부스트)
         ↓
  DAO 투표 (찬/반 + 배틀 결과 선택)
         ↓
  통과 → AI 영상 제작 파이프라인
         ↓
  ┌──────────────────────────────┐
  │  스테이킹 규모 → 영상 품질  │
  │  10K LORE  →  HD  1080p     │
  │  50K LORE  →  4K  Cinematic │
  │  200K LORE →  4K+ Dolby     │
  └──────────────────────────────┘
         ↓
  영상 공개 → 플랫폼 수익
         ↓
  승리 측 스테이커에게 보상 배분
```

### 1. Canon Lore (세계관 확장)

```
제안자 스테이킹 → Lorekeeper 검토 → DAO 투표
                                           ↓
                               통과 시: AI 시나리오 영상 제작
                               부결 시: 스테이킹 반환

영상 품질 = f(제안 스테이킹 + 커뮤니티 추가 스테이킹)
```

| 영상 품질 | 필요 스테이킹 | 사양 |
|-----------|---------------|------|
| SD | ~0 LORE | 720p · 30fps · 기본 |
| HD | 10,000 LORE | 1080p · 60fps · 보이스 + BGM |
| 4K | 50,000 LORE | 4K · Cinematic · 캐릭터 커스텀 |
| 4K+ | 200,000 LORE | 4K Dolby · 확장판 · NFT 에디션 |

### 2. Battle Arena (대전 투표)

드래곤볼 천하제일무술대회처럼 — 캐릭터 대전 결과를 커뮤니티가 결정하고 AI가 영상으로 구현.

```
창립팀/커뮤니티가 배틀 매치업 생성
            ↓
    커뮤니티가 LORE를 A팀 or B팀에 스테이킹
            ↓
    투표 마감 → 과반수 측이 승리
            ↓
    AI가 해당 결과의 배틀 영상 제작
            ↓
    ┌─────────────────────────────────┐
    │ 총 스테이킹량 → 영상 품질 결정 │
    └─────────────────────────────────┘
            ↓
    승리 측 스테이커: 패배 측 스테이킹의 80% 배분
    패배 측 스테이커: 20%는 플랫폼 Treasury로
    총 스테이킹의 5%는 영상 제작비로 소각
```

**Battle 유형:**
- 1v1 캐릭터 대전 (Kael vs. 아르코스)
- 세력 전쟁 (수호자 vs. 탐구자)
- 공연/콜라보 대결 (누가 더 개쩔것인가)
- 스토리 분기 투표 (이 캐릭터는 어떤 선택을 할 것인가)

---

### 3. Personal Scenario (개인 맞춤 시나리오)

**Phase 2+ 기능** — 자신이 원하는 시나리오를 스테이킹으로 제작 의뢰.

```
사용자 시나리오 입력
        ↓
LORE 스테이킹으로 제작 의뢰
        ↓
AI 영상 생성
        ↓
NFT로 민팅 (소유권 인증)
```

| 시나리오 유형 | 최소 스테이킹 | 비고 |
|---------------|---------------|------|
| 일반 (액션/판타지/드라마) | 5,000 LORE | 공개 배포 |
| 프리미엄 (고품질 개인 의뢰) | 20,000 LORE | 개인 소장 or 공개 선택 |
| 성인 콘텐츠 (Adult+) | 50,000 LORE | KYC 인증 + Adult 티어 필수 |

**Adult+ 티어 접근 조건:**
1. 나이 인증 (KYC 완료)
2. 5,000 LORE 스테이킹 (성인 멤버십)
3. 지역별 법령 준수 자동 필터링
