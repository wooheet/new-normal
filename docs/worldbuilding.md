# 세계관 설계 — OTR Universe

> 이 문서는 커뮤니티가 확장 가능한 세계관 프레임워크를 정의합니다.
> 핵심 설정(Core)은 창립팀이 정의하며, 나머지는 DAO 투표로 확장됩니다.

---

## 세계관 계층 구조

```
┌─────────────────────────────────────────┐
│  Tier 0 · CORE                          │  ← 절대 불변. 창립팀 전용.
│  세계의 근본 법칙, 창세 설정             │
├─────────────────────────────────────────┤
│  Tier 1 · CONFIRMED CANON               │  ← DAO 투표 75%+ 승인
│  주요 인물, 핵심 사건, 세력              │
├─────────────────────────────────────────┤
│  Tier 2 · EXTENDED CANON                │  ← DAO 투표 60%+ 승인
│  부수 인물, 장소, 아티팩트              │
├─────────────────────────────────────────┤
│  Tier 3 · OFF THE RECORD                │  ← DAO 투표 50%+ 승인
│  외전, 스핀오프, 평행 타임라인          │
├─────────────────────────────────────────┤
│  Tier 4 · FAN THEORY                    │  ← 커뮤니티 자유 토론
│  미확인 이론, 추측, 팬픽               │
└─────────────────────────────────────────┘
```

## 세계관 확장 가능 카테고리

커뮤니티가 제안하고 투표로 확장할 수 있는 카테고리 목록.
각 카테고리는 Tier에 따라 제안 가능 범위가 다릅니다.

### 1. 타임라인 / 역사 (Timeline)

```
[LOCKED - Tier 0]
- 세계의 시작점 (Year 0)
- 돌이킬 수 없는 대사건 (Turning Points)

[EXPANDABLE - Tier 1~3]
- 각 시대별 세부 사건
- 인물들의 과거 행적
- 발견되지 않은 역사적 사실
- 평행 타임라인 (OTR 전용)
```

**확장 규칙:**
- 새 사건은 기존 타임라인과 충돌하면 안 됨
- Tier 3 외전은 "본선에서 언급되지 않은" 시점만 다룰 수 있음
- 평행 타임라인은 Tier 3에서만 허용

---

### 2. 지리 / 공간 (Geography)

```
[LOCKED - Tier 0]
- 세계의 물리적 구조 (대륙, 차원, 우주 범위)
- 이동 불가능한 성역 또는 금지 구역

[EXPANDABLE - Tier 1~3]
- 새로운 도시, 마을, 지역
- 숨겨진 장소, 폐허
- 차원 내부 구조 (Tier 1)
- 미지의 영역 탐험 (Tier 2~3)
```

**확장 규칙:**
- 새 장소는 기존 지리 법칙(물리 법칙 포함)을 따라야 함
- "새로운 차원" 추가는 Tier 1 이상만 가능
- 지리 확장 시 해당 장소의 문화/세력도 함께 정의 권장

---

### 3. 세력 / 조직 (Factions)

```
[LOCKED - Tier 0]
- 세계를 움직이는 3~5개 핵심 세력
- 세력 간 근본 갈등 구조

[EXPANDABLE - Tier 1~3]
- 핵심 세력의 하위 조직
- 독립 세력, 중립 집단
- 비밀 결사, 지하 조직 (Tier 2~3)
- 소멸한 고대 세력 (Tier 3)
```

**확장 규칙:**
- 새 세력은 기존 세력 구도를 완전히 뒤엎을 수 없음 (Tier 1 투표 필요)
- 세력 제안 시 필수 기재: 목표, 수단, 핵심 인물, 약점

---

### 4. 인물 (Characters)

```
[LOCKED - Tier 0]
- 세계관의 축을 이루는 3~7인 핵심 인물
- 이들의 근본 동기와 운명

[EXPANDABLE - Tier 1~3]
- 주요 인물의 측근, 라이벌
- 독립적인 신규 캐릭터
- 핵심 인물의 과거 버전, 평행 세계 버전 (Tier 3)
- 커뮤니티 오리진 캐릭터 (Tier 2~3)
```

**확장 규칙:**
- 핵심 인물의 행동/동기를 바꾸는 설정은 Tier 1 필요
- 신규 캐릭터는 반드시 기존 세계와 연결점 1개 이상 보유
- 커뮤니티 캐릭터가 Canon 편입 시 LORE 토큰 보상 지급

---

### 5. 세계 법칙 / 시스템 (World Rules)

```
[LOCKED - Tier 0]
- 이 세계의 물리 법칙, 마법/기술의 근본 원리
- 생사 규칙, 시간 흐름 방식

[EXPANDABLE - Tier 1~2]
- 세계 법칙의 예외 케이스 (희귀하게만 허용)
- 새로운 마법/기술 분야 발견
- 미지의 능력 또는 아티팩트 원리
```

**확장 규칙:**
- 세계 법칙 관련 제안은 항상 Tier 1 이상 필요 (75%+ 투표)
- "예외"는 예외로 남아야 함 — 남발 시 거버넌스로 회수 가능
- 새 능력/기술 제안 시 반드시 한계(weakness)도 함께 정의

---

### 6. 아티팩트 / 오브젝트 (Artifacts)

```
[EXPANDABLE - Tier 1~3]
- 세계관을 바꿀 수 있는 강력한 유물 (Tier 1)
- 일반 마법 아이템, 기술 장비 (Tier 2)
- 소실된 유물, 전설 속 물건 (Tier 3)
```

**확장 규칙:**
- 강력한 아티팩트일수록 높은 Tier 투표 필요
- 아티팩트 제안 시 필수 기재: 능력, 한계, 소유자 조건, 역사

## 세계관 일관성 보호 장치

### Lorekeeper Council (세계관 수호자)
- LORE 토큰 상위 보유자 중 커뮤니티가 선출
- 새 제안의 Canon 충돌 여부를 1차 검토
- 충돌 발견 시 투표 전 커뮤니티에 경고 공지

### Core Veto
- 창립팀은 Tier 0 위배 제안에 대해 거부권 행사 가능
- 거부권 행사 시 반드시 공개 사유 제출 의무
- 거부권은 연간 최대 3회로 제한 (남용 방지)

### 설정 충돌 해결 프로세스
```
충돌 발생 감지
    ↓
Lorekeeper Council 검토 (3일)
    ↓
공개 토론 (5일)
    ↓
충돌 해소 방안 투표 (3일)
    (A안 채택 / B안 채택 / 두 설정 모두 Tier 3로 격하)
```
