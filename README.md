# OTR Universe
> Off The Record — 커뮤니티가 함께 만들어가는 세계관 프로젝트

---

## 프로젝트 개요

OTR Universe는 마블 유니버스처럼 방대한 세계관을 가진 영상 IP를,
**커뮤니티가 DAO 거버넌스로 함께 확장해나가는** 탈중앙화 스토리텔링 플랫폼입니다.

외전(Off The Record) 영상으로 시작해, 커뮤니티 반응과 투표에 따라
세계관이 살아 숨쉬며 진화합니다.

---

## 핵심 철학

- **Canon은 커뮤니티가 결정한다** — 어떤 설정이 공식이 될지 DAO 투표로 결정
- **외전이 정식이 될 수 있다** — Off The Record 콘텐츠가 Canon으로 편입 가능
- **기여한 만큼 목소리가 커진다** — LORE 토큰 기반 거버넌스

---

## 세계관 구조

```
[ Tier 0 ] Core Canon       ← 창립팀이 정의, 불변의 핵심 설정
[ Tier 1 ] Confirmed Canon  ← DAO 투표 75%+ 승인, 정식 설정
[ Tier 2 ] Extended Canon   ← DAO 투표 60%+ 승인, 공인 확장 설정
[ Tier 3 ] Off The Record   ← DAO 투표 50%+ 승인, 외전/스핀오프
[ Tier 4 ] Fan Theory       ← 미확인, 커뮤니티 토론 중
```

세계관 확장은 **제안 → 토론 → 투표 → 편입** 순서로 진행됩니다.

---

## LORE 토큰

OTR Universe의 거버넌스 토큰입니다.

| 용도 | 설명 |
|------|------|
| 투표권 | 거버넌스 투표 참여 (1 LORE = 1표) |
| 제안권 | Lore Proposal 제출 (최소 스테이킹 필요) |
| 보상 | 콘텐츠 기여, 거버넌스 참여 시 지급 |
| 접근권 | Tier별 멤버십 및 독점 콘텐츠 |

**총 공급량: 100,000,000 LORE**

```
창립팀          20% (4년 베스팅, 1년 클리프)
커뮤니티 트레저리 30%
콘텐츠 리워드    25%
얼리 컨트리뷰터  15%
리저브           10%
```

---

## 커뮤니티 참여 방법

1. **시청하기** — 유튜브/플랫폼에서 OTR 영상 감상
2. **토론하기** — Discord에서 세계관 토론 및 팬 이론 공유
3. **제안하기** — LORE 토큰 보유 후 세계관 확장 제안 제출
4. **투표하기** — 다른 커뮤니티 제안에 투표하여 세계관 결정
5. **제작하기** — 팬 콘텐츠 제작으로 LORE 토큰 보상 획득

## 로드맵 요약

| Phase | 기간 | 주요 목표 |
|-------|------|-----------|
| Phase 0 | 0~1개월 | 핵심 세계관 정의, Discord 오픈, 첫 OTR 영상 |
| Phase 1 | 1~3개월 | Lore Wiki 공개, 첫 거버넌스 투표, 커뮤니티 제안 영상화 |

| Phase 3 | 6개월+ | 풀 DAO, 외부 크리에이터 오픈, IP 확장 |
