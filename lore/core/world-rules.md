# OTR Platform — Universal World Rules

> **이 문서는 OTR Universe 플랫폼 자체의 작동 원리를 정의합니다.**
> 특정 IP(세계관)의 설정이 아닌, **어떤 IP든 이 플랫폼 위에서 작동하기 위한 메타-규칙**입니다.
> 창립팀만 수정 가능. DAO 투표로도 변경 불가.

---

## 1. 플랫폼의 존재 목적

OTR Platform은 **IP 소유자(기업/개인)와 커뮤니티가 함께 콘텐츠를 공동 생산하는 인프라**입니다.

- IP 소유자는 **Universe**를 등록하고 Tier 0 규칙(불변 설정)을 정의한다
- 커뮤니티는 그 위에서 **Proposal**을 제출하고 DAO로 투표한다
- 승인된 제안은 AI 파이프라인을 통해 **실제 콘텐츠(영상, 게임 요소 등)** 로 생산된다
- 생산 품질은 커뮤니티의 **스테이킹 규모**에 비례한다

---

## 2. Universe (유니버스)

**Universe = 하나의 IP를 담는 독립 컨테이너**

| 항목 | 설명 |
|------|------|
| `slug` | 고유 식별자 (예: `maplestory`, `one-piece`, `my-original-world`) |
| `owner` | Universe 등록자. IP 소유 기업 또는 개인 |
| `tier0_rules` | 해당 IP의 절대 불변 설정 (IP 소유자가 직접 정의) |
| `content_types` | 생산 가능한 콘텐츠 종류 (video / game_asset / webtoon 등) |
| `governance_threshold` | 제안 통과에 필요한 찬성 비율 (기본 51%) |

### Universe 독립성 원칙

- 각 Universe는 **서로 완전히 격리**된다
- Universe 간 크로스오버는 **양측 Universe 모두의 DAO 승인**이 있어야만 허용된다
- 한 Universe의 규칙이 다른 Universe에 영향을 줄 수 없다

---

## 3. Tier 계층 (모든 Universe에 공통 적용)

| Tier | 명칭 | 결정권 | 변경 가능 여부 |
|------|------|--------|---------------|
| **Tier 0** | Core Rules | IP 소유자 단독 | 불가 — 소유자만 수정 |
| **Tier 1** | Confirmed Canon | DAO 67% + Lorekeeper 3인 | DAO 투표 |
| **Tier 2** | Extended Canon | DAO 51% | DAO 투표 |
| **Tier 3** | OTR (Off The Record) | DAO 40% | DAO 투표 |

---

## 4. 콘텐츠 생산 파이프라인 (불변 원칙)

1. **Proposal 제출** → 커뮤니티가 시나리오/콘텐츠 아이디어를 제출
2. **Lorekeeper 검토** → Tier 0 충돌 여부 확인
3. **DAO 투표** → 스테이킹 가중 투표
4. **승인 시 큐(Queue) 진입** → AI 파이프라인으로 전달
5. **품질 결정** → 총 스테이킹 규모에 따라 SD / HD / 4K / 4K+ 자동 배정
6. **콘텐츠 배포** → Universe 내 커뮤니티에 공개

### 스테이킹 → 품질 매핑 (기준값, Universe별 커스터마이즈 가능)

| 품질 | 기본 스테이킹 기준 | 예시 콘텐츠 |
|------|-------------------|------------|
| SD | 0 LORE | 15초 스케치 영상 |
| HD | 10,000 LORE | 30초 시네마틱 |
| 4K | 50,000 LORE | 1분 풀 시네마틱 |
| 4K+ | 200,000 LORE | 3분+ 영화급 영상 |

---

## 5. 크로스포인트 이벤트 (Universe 간 크로스오버)

두 Universe가 공식 협업할 때 발생하는 특별 이벤트.

**발동 조건:**
- 양측 Universe의 DAO가 각각 67% 이상 찬성
- 양측 IP 소유자의 명시적 동의
- 크로스오버 Tier 0 충돌 없음 확인

**결과물:**
- 크로스오버 전용 별도 Universe 또는 공동 Canon 항목 생성
- 양측 커뮤니티에 동시 배포

---

## 6. 플랫폼 금지 사항 (모든 Universe 공통)

어떤 Universe의 어떤 Proposal도 다음을 포함할 수 없다:

- IP 소유자의 Tier 0를 직접 수정하는 제안
- 실존 인물을 특정하는 콘텐츠
- 다른 Universe의 콘텐츠를 무단 도용
- 성인 전용 콘텐츠 (해당 Universe가 Adult 허용으로 명시적 설정하지 않은 경우)
